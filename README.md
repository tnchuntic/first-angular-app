

# My First Angular App

----------


# Overview
This is my first angular app repo. This is intended to be a starting template for angular projects.


# Dependencies
* `Node > 6.x`
* `npm > 3.x`
* globals (`npm install --global`)
    * `typescript`
    * `webpack`
    * `webpack-dev-server`
    * `tslint`


# Getting Started

- Clone or fork this repository
- run `npm install --global typescript webpack webpack-dev-server tslint`
- run `npm install`
- run `npm start` to fire up dev server
- open browser to [`http://localhost:3000`](http://localhost:3000)

# Workflow
* Start the course 
* Watch the video for the current lesson
* Checkout to the appropriate branch for the lesson
* Follow along and build the app!
